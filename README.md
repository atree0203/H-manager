## vue + elementUI 创建后台管理系统框子


### 下载项目
1. 使用脚手架
    ```
    npm install vue-element-manager-cli -g
    vue-element-manager-cli init <projName>
    ```
2. 直接下载
    ```
    git clone https://github.com/PanJiaChen/vue-element-admin
    ```

### 构建项目
```
cd <projName>
npm install
npm run serve
```

### 编译压缩
```
npm run build
```
